# Project Plan — Customer Support Automation using AI

## Objective
To build an AI-based customer support system that can answer frequently asked questions and escalate complex issues, improving customer service efficiency.

## Project Timeline (Simple and Easy)
### Week 1 — Research & Requirements
- Write sample customer questions and answers
- Collect real call/chat examples if available

### Week 2 — Create Sample Dataset
- List 10–20 common support queries and responses

### Week 3 — Create Demo Flow
- Show how the AI agent responds
- Add example conversation steps

### Week 4 — Presentation Slides
- Prepare PPT for project explanation

### Week 5 — Testing & Feedback
- Share demo with 3–5 people and collect feedback

### Week 6 — Final Publishing
- Add final report & PPT into GitHub

## Expected Outcome
A functional AI support example that shows how customer queries can be handled without a human agent.

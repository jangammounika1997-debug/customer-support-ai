# Customer Support Automation using AI

## Project Overview
This project focuses on automating customer support tasks using AI.  
The goal is to reduce response time and improve customer satisfaction by enabling AI to answer common questions automatically and escalate complex issues to a human agent.

## Key Features
- AI answers customer questions using sample FAQs
- Escalates complex queries
- Reduces workload on human agents

## Deliverables
- Project plan
- Sample FAQ dataset
- Working demo explanation
- Final PPT presentation
